⚙️ Nạp file thủ công (kiểm tra / dự phòng)

Dùng mục này để tự nạp file kiểm tra trước khi đưa lên GitHub, hoặc khi liên kết dùng chung gặp lỗi.

Excel: mỗi sheet = 1 tuyến kênh (tên sheet = tên kênh). Cột cần có: K bắt đầu, K kết thúc, Người phụ trách.

1 · Lớp hiển thị
Nạp file KML/KMZ để xem danh sách kênh, cống, trạm theo đúng cấu trúc thư mục trong file của bạn.
2 · Công cụ đo
Chọn một công cụ, click các điểm trên bản đồ. Click đúp hoặc bấm Enter để kết thúc, Esc để hủy.
Bấm vào một tuyến kênh hoặc một điểm công trình trên bản đồ — thông tin chi tiết sẽ hiện trong khung lớn phía dưới bản đồ.
Phiên bản thử nghiệm chạy hoàn toàn trên trình duyệt — dữ liệu nạp vào sẽ mất khi tải lại trang. Mỗi phiên làm việc, hãy nạp lại file KML/KMZ và Excel gốc của đơn vị. Định vị GPS (nút ◎ trên bản đồ) cần mở file qua máy chủ HTTPS (vd: lưu trên Google Drive/SharePoint rồi mở link, hoặc host nội bộ) — mở trực tiếp file trên máy (file://) thường bị trình duyệt từ chối quyền định vị.
